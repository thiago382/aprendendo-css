Tutorial de CSS

CSS é a linguagem que usamos para estilizar um documento HTML. O CSS descreve como os elementos HTML devem ser exibidos. Este tutorial ensinará CSS do básico ao avançado.

Exemplo de CSS body { background-color: lightblue; } h1 { color: white; text-align: center; } p { font-family: verdana; font-size: 20px; }.
